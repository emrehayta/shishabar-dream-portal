# Taxim Shisha Bar – Website

Website für die [Taxim Shisha Bar](https://taximloungewels.at) in Wels, Österreich.

**Stack:** React · TypeScript · Vite · Tailwind CSS · shadcn/ui  
**Deployment:** GitHub Pages via GitHub Actions (push auf `main` → automatisch live)

```sh
npm install
npm run dev
```
